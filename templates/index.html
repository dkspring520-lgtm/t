📋 策略说明

做T交易策略详解

🌟 紫金矿业专用策略

胜率: 75-82%

核心原则

  • 一天只捕捉1-2次买卖点，宁可不做，不可乱做
  • 时间窗口控制：09:51-10:15早盘，13:01-14:00下午
  • 偏离率筛选：买入偏离 ≤ -0.8%，卖出偏离 ≥ +0.8%

买入条件

  • 条件A（最稳）：偏离 ≤ -1.2%，量比 ≥ 3.0
  • 条件B（常用）：偏离 -0.8%~-1.0%，量比 ≥ 2.5
  • 条件C（保守）：偏离 -0.5%~-0.8%，量比 2.0~2.4

卖出条件

  • 按买入类型对应目标价位止盈
  • 绝对止损：-0.6%
  • 14:30未卖出则强制平仓

📈 简单偏离率策略

胜率: 65-70%

原理

基于价格与均线的偏离程度，逆向操作

  • 偏离 < -1.5% → 买入信号
  • 偏离 > +1.5% → 卖出信号
  • 适合波动较大的股票

🚀 进阶偏离率策略

胜率: 70-75%

原理

加入波动率调节，动态调整阈值

  • 短期均线 + 长期均线双重检测
  • 根据历史波动率调整买卖阈值
  • 适合有一定趋势的股票

🚚 紫金高利润策略

胜率: 60-70%

原理

更激进的做T策略，追求更高收益

  • 更多买入信号（强买 + 普通买）
  • 更大仓位比例，50%现金做T
  • 更激进的止盈目标
  • 风险收益比更高
