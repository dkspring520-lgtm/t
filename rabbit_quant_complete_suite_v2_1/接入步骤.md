# 接入步骤

1. 把 `backend/` 复制到现有 Python 后端；
2. 把 `frontend/*.js` 和 `frontend/assets/` 复制到网站静态目录；
3. 将行情接口映射为：日线、完成的1分钟K线、09:15—09:25竞价快照；
4. 后端调用 `build_complete_payload()`；
5. 增加 `/api/rabbit/complete` 接口返回 JSON；
6. 操盘台内挂载 `RabbitQuantComplete`；
7. 09:25更新竞价预判，09:35再次更新验证版，之后每5分钟刷新盘中研判；
8. 收盘后执行自成长模块 `end_of_day()`；
9. 每周生成 Challenger，默认人工确认后再晋级。

## 放在现有操盘台哪里

- 左侧导航可增加“走势研判”；
- 当前股票主图右侧或顶部下方放紧凑卡片；
- 集合竞价卡放在研判卡最上方；
- 不要覆盖分时主图；
- 确认信号用右下角兔兔提醒卡展示。
