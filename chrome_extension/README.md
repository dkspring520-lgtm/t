# A股做T盯盘助手 - Chrome扩展

## 功能概述

专为A股做T设计的Chrome扩展，提供实时监控、买卖点提醒和模拟测试功能。

## 功能特点

### 1. 实时监控
- 自动获取A股实时行情（东方财富API）
- 3秒刷新价格数据
- 分时图展示分时走势
- 交易时间自动检测（9:30-11:30, 13:00-15:00）
- 休市状态提示

### 2. 智能买卖点提醒
- MACD金叉/死叉检测
- 布林带上下轨触发
- RSI超买超卖判断
- VWAP支撑压力分析
- 日内图上标注买卖点（买-绿色圆点，卖-红色圆点）
- 弹窗+通知提醒

### 3. 持仓管理
- 设置持有股票名称、数量、成本价
- 最大做T仓位控制
- 按持仓状况智能推荐买卖

### 4. 热门股池
- 自动获取市场上10只热门股票
- 默认包含：贵州茅台、宁德时代、比亚迪等高流动性股票
- 点击热门股即可自动填充到测试框

### 5. 批量模拟测试
- 一键测试10只热门股的做T策略胜率
- 实时进度条显示
- 每只股票显示胜率、总收益、赢亏次数
- 总体统计平均胜率和总收益

### 6. 单股模拟测试
- 输入任意股票代码测试
- 可设置测试资金
- 可选测试周期（1天/5天/10天/20天）
- 完整交易记录输出


## 安装步骤

### 开发者模式加载

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展"
4. 选择 `chrome_extension` 文件夹
5. 扩展已安装，点击工具栏图标使用

## 使用说明

### 实时监控
1. 点击扩展图标打开弹窗
2. 输入股票代码（如600519）
3. 点击"加载"获取行情
4. 填写持仓信息（可选）
5. 点击"开始监控"
6. 等待买卖点提醒

### 热门股池功能
1. 切换到"模拟测试"标签
2. 点击"获取当前热门股票" - 将加载市场上流动性好的10只股票
3. 点击任意热门股，自动填充到测试框

### 一键批量测试
1. 设置测试资金（默认10万）
2. 选择测试周期（1天/5天/10天/20天）
3. 点击"批量测试10热股"
4. 等待测试完成，查看每只股票的胜率和收益
5. 底部显示总体统计结果

### 单股模拟测试
1. 切换到"模拟测试"标签
2. 输入测试股票代码
3. 设置初始资金和测试周期
4. 点击"单股测试"
5. 查看胜率和交易记录

## 数据来源

- 实时行情：东方财富API (`push2.eastmoney.com`)
- 分时数据：东方财富分时接口
- 热门股票：东方财富榜单API

## 注意事项

1. 扩展需要网络连接获取实时数据
2. 监控只在交易时间内有效（9:30-11:30, 13:00-15:00）
3. 买卖点仅供参考，不构成投资建议
4. 建议先使用模拟测试验证策略效果
